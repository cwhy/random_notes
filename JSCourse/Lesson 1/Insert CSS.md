# Three ways to insert CSS into HTML

- insert a whole CSS file
```html
<head>
	<link rel="stylesheet" href="styles.css">
</head>
```

- insert CSS into HTML
```html
<style>  
	body {
		background-color: powderblue;
	}  
	h1   {
		color: blue;
	}  
	p    {
		color: red;
	}  
</style>
```

- insert CSS inline 
```html
<p style="color:red;">A red paragraph.</p>
```
