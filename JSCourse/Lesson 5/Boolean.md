## Boolean
```js
1 === 1 // true
1 === 2 // false
1 !== 2 // true
1 !== 1 // false
// !(a === b) === (a !== b)
```

Logical Operations

```js
// And
true && true // true
true && false // false
(1 === 1) && ("21" === "21") && (12 === "12")// false

// Or
true || false // true
false || false // false

// Not
!true // false
!100 //false
```
