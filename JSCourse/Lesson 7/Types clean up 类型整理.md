## Types
When you see a `x`
- numbers 数字
	```js
	let x = 1
	```
- strings 字符串
	```js
	let x = "hello, 1"
	```
- array 数组
	``` js
	let x = [2, 3, 4]
	```
- boolean 布尔值
   ```js
   let x = true
   ```
- functions 函数
	```js
	let x = y => y + 1
	```

## Function input and output types

Input and output type of functions can be one of the types.

- `alert` inputs string, outputs nothing,
but there will be side effects.
- `console.log` inputs string, outputs nothing
- `x => x + 1`
- `x => [x, 1, 2]`
- `[2, 3, 4].map` inputs
- `[2, 3, 4].filter` inputs
